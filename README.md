# Student Result Management System

## Software Development Life Cycle (SDLC)

### 1. Planning
The goal is to develop a system that allows schools to record, manage and check studentsâ€™ results online.

### 2. Requirement Analysis
The system should:
- Allow teachers to upload results
- Allow students to check results
- Store student records
- Provide secure login

### 3. System Design
The system will have:
- Login page
- Result upload page
- Result checking page
- Database to store records

### 4. Implementation
The system is developed using HTML to create the interface.

### 5. Testing
The system was tested to ensure:
- Results display correctly
- Login works
- Pages load properly

### 6. Deployment
The project is uploaded to GitHub for public access.

### 7. Maintenance
Future updates can include database and security improvements.
